# nfo/ — the sidecars

93 ready-to-use Kodi/XBMC `<movie>` `.nfo` metadata sidecars for the official
AARO / PURSUE UAP videos, grouped by release:

- `release-01/` — 28 (Release 01, May 8 2026)
- `release-02/` — 56 (Release 02, May 22 2026)
- `release-03/` — 9  (Release 03, June 12 2026)

Each file is named `DOD_<id>.nfo` to match the DVIDS download `DOD_<id>.mp4`.
Drop a sidecar next to its matching video (same folder, same basename) and turn
on NFO reading in your media server. See the top-level `README.md` for the full
walkthrough and `data/catalog.csv` for the DVIDS download link per video.

Titles and descriptions are verbatim from the videos' embedded metadata
(Release 01) and official DVIDS pages (Releases 02–03). Nothing is invented.
